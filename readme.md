# Vinyl Record Portfolio Website – Project README / Claude Code Context

## Project Overview

This project is a **premium creative portfolio website** for **Mariam Lokhat**, inspired by vinyl records and vintage record players.

The goal is **not** to build a gimmick—it should feel like an award-winning Awwwards / Studio Freight / Locomotive style portfolio that happens to use a vinyl metaphor.

Think:

* Apple product page quality
* Spotify + Criterion Collection
* Kinfolk / Monocle editorial layouts
* High-end creative agency portfolio

Every interaction should feel intentional and premium.

---

# Core Concept

The homepage is a vintage record player.

Five records represent five sections of Mariam's portfolio.

```
A1 About Me
A2 Copywriting
A3 Concepts
A4 Sound Design
A5 Film
```

Each record is selectable.

When clicked:

1. Record lifts
2. Slides onto platter
3. Tonearm moves
4. Record spins
5. Page transitions into that portfolio section

No hard page reloads.

Everything should feel physical.
